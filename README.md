### 项目介绍 ###
这是一个最小的IEC61499 项目，使用Python 编写了IEC61499 功能块，功能块网络和运行时的类。以及几个功能块类型的实现。
本项目的并不是开发一个Python 的IEC61499 runtime，而是研究python 作为一种信息模型描述语言，在AI Code 中的效果如何。
本项目使用VSCode+Claude Code+GLM4.5 实现。底层class 是人工编写的。
在此基础上可以将4idac 项目中的大量功能块库转换成为python 功能块库。也可以转换成为C++ 类。
最终目标是使用Claude Code 实现IEC61499 的全部开发工作。不再依赖图形化组态编写IEC61499 的应用。推动IEC61499 进入AI Code时代。
同时，我们研究Python 作为一种系统工程中的模型语言，开发各种工程模型的生成。包括OPC UA，I4.0 AAS，PLCOpen ，P&ID 等。
