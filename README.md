### 项目介绍 ###
这是一个最小的IEC61499 项目，使用Python 编写了IEC61499 功能块，功能块网络和运行时的类。以及几个功能块类型的实现。
本项目的并不是开发一个Python 的IEC61499 runtime，而是研究python 作为一种信息模型描述语言，作为与AI Code 交流的语言。
本项目使用VSCode+Claude Code+GLM4.5 实现。底层class 是人工编写的。在此基础上可以将4idac 项目中的大量功能块库转换成为python 功能块库。也可以转换成为
C++ 类。
项目的最终目标是使用Claude Code 实现IEC61499 的全部开发工作。不再依赖手工图形化编写IEC61499 的应用。基于基本的库和工具，推动IEC61499 进入AI时代。
同时我们研究Python 作为一种系统工程中的模型语言，在Ai系统工程中发挥的作用，包括OPC UA，I4.0 AAS，PLCOpen ，P&ID 等应用工程中应用的可能性。
